# qbi-dev-tools
This is a building tool for Quick-BI developers.

## Install

with yarn:
```
yarn add -D qbi-dev-tools
```

with npm:
```
npm i -D qbi-dev-tools
```
## starting dev server

```
npm run start
```

## building the project
```
npm run build
```

## packing the project
```
npm run pack
```
